# Beauty Salon - Booking Management System

Web application to manage services, users and bookings for a beauty salon.
Built with **PHP (custom MVC)**, **MySQL**, **JavaScript**, **SCSS** and **Gulp**.

## Features

- User registration, email confirmation and login
- Password recovery via token
- Services management (CRUD)
- Booking creation with service selection, date and time
- Admin panel to view and delete bookings
- Asset pipeline with Gulp and image optimization
- Email notifications using PHPMailer

##  Technologies

- **PHP 8+**
- **Composer**
- **MVC Personalizado**
- **MySQL / MariaDB**
- **JavaScript**
- **Sass**
- **Gulp**
- **PHPMailer**
- **Dotenv**
- **SweetAlert2**
